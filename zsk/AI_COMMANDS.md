# AI 命令参考

将此文件中的命令复制粘贴到 Hermes 对话框中即可。

## 场景 1：首次构建 / 完整重建（最常用）

加载 zsk-build skill，然后构建知识库。

## 场景 2：增量更新（已有知识库，新增了研报）

加载 zsk-build skill。运行 kb.py build 获取分析报告，只新增不存在的概念节点，已有概念用 kb.py edit 追加内容，相同的概念合并到一个节点下。完成后执行 reorganize、dedup、export。

## 场景 3：研报格式不规范（无标题层级）

加载 zsk-build skill。reports 目录下的研报格式不标准，请直接阅读每份研报的完整内容，做语义理解和概念提取。将提取的概念按 8 个本体分类归类，相同的概念合并。最后 reorganize、dedup、export。

## 场景 4：搜索查询

搜索知识库里关于 MCP 的内容。

## 场景 5：导出可视化

导出知识库可视化。

## 场景 6：修复杂乱的知识树

重建分类树并去重合并，然后导出。
