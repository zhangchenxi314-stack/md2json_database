# zsk 功能概要

## 输入

- 多份 Agent 技术 Markdown 研报（放 `reports/` 目录）

## 输出

- **JSON 知识库**（结构化，人和 AI 均可读写）
- **HTML 知识树**（浏览器打开，可交互浏览）

## 核心功能

- **研报解析** — 按标题层级自动提取知识点，生成结构化节点
- **本体分类** — 每个知识点归入 8 大本体分类（架构/规划/工具/RAG/记忆/多Agent/评估/安全）
- **智能合并** — 多份研报中相同概念自动归并，内容叠加，避免重复
- **标题归一化** — `"AI Agent 工具调用"` 和 `"工具调用"` 识别为同一概念
- **四层知识树** — 主根 → 分类 → 概念 → 子概念，层级清晰
- **Agent 智能构建** — 由 LLM 语义理解研报内容，精准分类和合并，超越规则匹配
- **交互式 HTML** — 点击展开/折叠节点、搜索过滤、优先级色标、Markdown 完整渲染
- **跨平台一键运行** — Windows 双击 `build.bat`，macOS/Linux 运行 `build.sh`
- **跨机器移植** — 复制项目到任意电脑，`python kb.py setup` 即可接入 Hermes Agent
- **命令行管理** — 搜索、增删改查、统计、导出，14 个 CLI 命令
- **最小依赖** — 仅需 `markdown` 一个库（纯 Python），其余全用标准库
